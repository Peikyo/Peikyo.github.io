周报

——第二次

1. 本周学习方向

树状数组
链表详细操作

2. 学习参考

树状数组：https://blog.csdn.net/bestsort/article/details/80796531

3. 总结

树状数组利用二进制操作，用类似于二叉树把数组操作的复杂度降了很多，主要用于数组的单点修改与区间求和等问题，但我练习的例题还不够多，对于它的更高级操作还不会运用，下周进一步学习。
这周上课老师详细讲解了链表的创建、查找、插入、删除、释放等操作，这几种操作基本都是对于链表节点指针域的操作，因为是基础操作所以比较而言相对简单。做题时也发现链表可以和其它的如stack函数结合起来，达到更高效解题的效用。

4. Leetcode

题目1：合并两个有序链表
思考：对两个链表的值进行逐一比对插入，最后多余的连在后面
题目2：回文链表
思考：将链表地址压入栈中，再逐个进行对比
题目3：环形链表
思考：设立快指针与慢指针，如果有环，慢指针会与快指针相遇

5. 下周进度

树状数组、背包、单调栈补题
