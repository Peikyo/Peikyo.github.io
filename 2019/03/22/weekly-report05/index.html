2. 学习参考

3. 总结

二维数组指针的学习点在于函数参数的传递，传递时必须明白数组指针与指针数组的本质上的差异，另外便是二维数组与一维数组在设置形参的时候有稍许不同，有一些强制注意事项，如二维指针的第二维在设置形参时必须明确长度。
动态规划以时间换空间的原理，降低算法的复杂度，背包问题是其中一种典型情况。通过对每一重量每一体积的数据进行计算存储，方便了以后情况的数据调用，从而达到了减少时间的目的。
链表的学习现在还在初步阶段，具体使用已经明白了，但是链表的创建反而有些不明白，具体的等着老师下一次课具体讲解。

4. Leetcode

题目1：删除链表中的节点
思考：直接用它的下一个节点替换该节点
题目2：删除链表的倒数第N个节点
思考：一次搜索后计出节点个数，再二次搜索删除节点
题目3：反转链表
思考：将每个结点的指针域压入栈中，再直接赋值改动，空链表应该特别用条件分出
